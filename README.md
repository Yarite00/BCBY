# Инструкция по установке
1) Заходишь в [CurseForge приложение](https://download.overwolf.com/install/Download?ExtensionId=cfiahnpaolfnlgaihhmobmnjdafknjnjdpdabpcm&utm_term=eyJkb21haW4iOiJjZi13ZWIifQ%3D%3D)
2) На главной странице Minecraft (где лежат все твои сборки) будет кнопка Import - На нее нажимаешь
3) Выбираешь Import Profile .zip
4) В открывшемся окне открываешь архив с гитхаба
5) Все, сборка скачает сама себя
